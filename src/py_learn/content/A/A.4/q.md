# 字符串操作

你有没有试过从一堆名字中快速地认出第一个字和最后一个字？就像在一排队伍里能立刻指出排头和排尾一样，Python 的字符串索引就是做这个的。

## 概念讲解

字符串是由一个个字符组成的序列。每个字符都有自己固定的位置编号，这个编号就是**索引（index）**。

Python 的索引从 **0** 开始——这一点和生活中从 1 开始数数的习惯不同，需要特别适应。

把字符串 `"Python"` 想象成一条有 6 个格子的带子：

```
字符:    P   y   t   h   o   n
正索引:  0   1   2   3   4   5
负索引: -6  -5  -4  -3  -2  -1
```

正索引从左往右数，从 0 起步；负索引从右往左数，从 -1 起步。同一位置有两种编号方式：`P` 既是 `[0]` 也是 `[-6]`，`n` 既是 `[5]` 也是 `[-1]`。

**len() 函数**返回字符串中字符的个数。注意：`len()` 数的是"有几个"，结果是 1 起步的计数（有 6 个就是 6），而索引是 0 起步的位置号——最后一个字符的索引是 `len() - 1`。

## 语法拆解

**正索引——从左往右取：**

```python
word[0]    # 第一个字符
word[1]    # 第二个字符
word[2]    # 第三个字符
```

中文也一样：

```python
"你好世界"[0]    # "你"
"你好世界"[1]    # "好"
```

**负索引——从右往左取：**

```python
word[-1]   # 最后一个字符
word[-2]   # 倒数第二个字符
word[-3]   # 倒数第三个字符
```

**len()——获取长度：**

```python
len("Python")     # 返回 6
len("你好")        # 返回 2（中文每个字算一个字符）
len("a")          # 返回 1
```

**索引与 len 的关系：**

```python
# 最后一个字符的正索引 = len(word) - 1
word[len(word) - 1]   # 等价于 word[-1]
```

**f-string 格式化：**

```python
f"首尾字母：{word[0]}{word[-1]}，长度：{len(word)}"
```

## 常见错误

1. **索引从 1 开始**：初学者常以为 `word[1]` 是第一个字符，实际它是第二个。第一个字符永远是 `word[0]`。

2. **索引越界**：索引不能超过 `len(word) - 1`。

```python
word = "abc"
word[3]   # 错误！只有索引 0、1、2，3 越界了
```

3. **对空字符串取索引**：

```python
""[0]    # IndexError！空字符串没有字符，谈不上"第一个"
```

4. **混淆 len 和索引**：`len("abc")` 返回 `3`（有 3 个字符），但最后一个字符的索引是 `2`，不是 `3`。因为索引从 0 开始，3 个字符的索引范围是 0~2。

## 题目要求

编写函数 `def py_learn(word)`，接收一个字符串 `word`，返回一个格式化字符串：

```
首尾字母：{首字母}{尾字母}，长度：{长度}
```

示例：
- `py_learn("Python")` → `"首尾字母：Pn，长度：6"`
- `py_learn("你好")` → `"首尾字母：你好，长度：2"`
- `py_learn("a")` → `"首尾字母：aa，长度：1"`（单字符时首尾相同）

## 提示

- 用 `word[0]` 获取首字母
- 用 `word[-1]` 获取尾字母
- 用 `len(word)` 获取长度
- 用 f-string 格式化输出结果：`f"首尾字母：{...}{...}，长度：{...}"`
