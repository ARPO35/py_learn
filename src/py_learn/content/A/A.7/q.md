# 练习 A.7：条件判断 if/else

## 生活类比

想象你去游乐园，门口立着一个身高标尺：**超过 1.4 米才能玩过山车**。工作人员测量后，要么放行，要么拦下——代码里也需要这种「根据不同情况走不同路线」的能力，这就是 **条件判断**。

## 概念讲解

`if` 语句让程序根据条件的成立与否（True / False）决定执行哪段代码。基本结构：

```python
if 条件:
    条件成立时做的事
else:
    条件不成立时做的事
```

- **条件**：通常是一个比较表达式（如 `age >= 18`），结果要么是 `True`，要么是 `False`
- **冒号 `:`**：`if` 和 `else` 后面**必须**跟冒号，告诉 Python 后面是代码块
- **缩进**：属于 `if` 或 `else` 的代码必须缩进（通常 4 个空格），Python 靠缩进识别代码块归属

## 代码示例

```python
# 示例 1：成绩判断
score = 85
if score >= 60:
    print("及格")
else:
    print("不及格")
# 输出：及格
```

```python
# 示例 2：数字奇偶判断
num = 7
if num % 2 == 0:
    print("偶数")
else:
    print("奇数")
# 输出：奇数
```

## 常见错误

1. **忘记冒号**：`if age >= 18`（少写 `:`）→ `SyntaxError`
2. **缩进不一致**：同一代码块有的缩进 4 格，有的缩进 2 格 → `IndentationError`

## 题目要求

编写函数 `py_learn(age)`，接收一个整数 `age`（年龄）。函数按以下规则返回字符串：

- 如果 `age >= 18`，返回 `"已成年，可以进入"`
- 否则，返回 `"未成年，不可进入"`

## 提示

- 函数签名：`def py_learn(age):`
- 用 `if age >= 18:` 作为判断条件
- 两个分支各写一个 `return` 语句
- 注意 `if` 和 `else` 后面的冒号，以及代码缩进
