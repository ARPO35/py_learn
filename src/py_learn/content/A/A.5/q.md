# 练习 A.5：while 循环求阶乘

## 题目说明

编写函数 `py_learn`，接收一个非负整数 `n`，使用 `while` 循环计算 `n` 的阶乘（n!）并返回。

阶乘定义：
- n! = n × (n-1) × (n-2) × ... × 1
- 0! = 1

## 示例

```python
py_learn(5)    # 返回 120（5 × 4 × 3 × 2 × 1）
py_learn(3)    # 返回 6（3 × 2 × 1）
py_learn(0)    # 返回 1
```

## 提示

- 使用 `while` 循环：`while n > 0:`
- 初始化 `result = 1`
- 每次循环让 `result *= n`，然后 `n -= 1`
- 注意 0! = 1 这个特殊情况

## 要求

- 函数签名：`def py_learn(n):`
- 必须使用 `while` 循环实现
- 返回值类型：`int`
