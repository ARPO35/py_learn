# 练习 A.5：类型转换

## 生活类比

快递单上的「电话」是文字还是数字？快递员看到 "13812345678" 时，这不是一个可以计算的数字，而是一串字符（就像门牌号不会用来做加减法）。Python 里的数据也有「身份」：有些看起来像数字，实际是字符串；有些确实是整数。

## 概念讲解

每种数据都有类型，用 `type()` 可以查看：

```python
type(42)      # <class 'int'>    整数
type("42")    # <class 'str'>    字符串
type(3.14)    # <class 'float'>  浮点数
```

Python 不会自动把字符串当数字用。要让字符串里的数字参与计算，需要「类型转换」：

- `int("42")` — 把字符串转成整数 → `42`
- `float("3.14")` — 把字符串转成浮点数 → `3.14`
- `str(42)` — 把整数转成字符串 → `"42"`

## 语法拆解

### 查看类型

`type()` 用来检查数据的"身份"——就是几行孤立的演示，不需要连成完整程序：

```python
type(42)      # <class 'int'>    整数类型
type("42")    # <class 'str'>    字符串类型
type(3.14)    # <class 'float'>  浮点数类型
```

注意：`"42"` 和 `42` 长相相似，但类型完全不同——一个被引号包裹，一个没有。

### 字符串拼接 vs 数字加法

同样的 `+` 号，对不同的类型做完全不同的事：

```python
# 两个字符串用 + → 拼接（粘在一起）
"42" + "10"   # 结果是 "4210"
```

```python
# 两个整数用 + → 数学加法
42 + 10   # 结果是 52
```

把这两行放在一起对比，就能看出类型决定了运算的含义。

### 类型转换：int() 和 str()

如果要把字符串里的数字当整数计算，必须先用 `int()` 转换：

```python
int("42")   # → 42（字符串变成了整数）
```

反过来，把整数嵌入到字符串中时，需要 `str()`：

```python
str(42)   # → "42"（整数变成了字符串）
```

### 格式化输出：f-string

`f"..."` 是 Python 的格式化字符串，花括号 `{}` 里可以写变量或表达式，Python 会自动把结果嵌入到字符串中：

```python
f"结果：{52}"       # → "结果：52"
f"{42 + 10}"        # → "52"（花括号里可以直接写表达式）
```

### 分步思路

完成类型转换+计算+格式化的思路，每一步都是独立的：

```python
# 第 1 步：拿到字符串，转成整数
# 例如 "42" 用 int() 变成 42
```

```python
# 第 2 步：做加法
# 42 + 10 得到 52
```

```python
# 第 3 步：格式化成目标字符串
# f"结果：{52}" 得到 "结果：52"
```

三步各是各的，学习者需要自己把它们串起来。

## 常见错误

1. **转换无法识别的内容**：`int("abc")` 或 `int("3.14")` 会报错 `ValueError`，因为"abc"不是合法的整数表示，"3.14"是浮点数格式而非整数格式。
2. **忘记转换**：直接用 `"42" + 10` 会直接报错 `TypeError`，因为字符串和整数不能直接相加。

## 题目要求

编写函数 `py_learn(num_str)`：
- 接收一个字符串参数 `num_str`（例如 `"42"`）
- 将其转换为整数，加 10，再格式化为字符串
- 返回格式：`"结果：52"`（注意：冒号是英文冒号，后有空格）

2. 做加法运算
