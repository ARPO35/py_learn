# 你好，Python！

第一次接触编程，就像走进一个陌生的国度，你需要学习怎么和当地人打招呼。对于 Python 这门语言来说，最简单的打招呼方式就是 **`print`**。

## 概念讲解

`print` 是 Python 自带的一个"函数"。你可以把它想象成一个扬声器——你把想说的话放在它后面的括号里，它就会把这些话显示在屏幕上，让你看到结果。

基本语法结构是这样的：

```python
print(要显示的内容)
```

括号里放的就是你想让计算机"说"的话。注意 `print` 全部小写，前后括号必须配对。

## 语法拆解

### 字符串：用引号包裹文本

你想让计算机显示的文本，在 Python 中叫做**字符串**。字符串需要用引号包裹起来，这样 Python 才能区分"这是要显示的文字"和"这是代码指令"。

```python
# 用双引号包裹
"这是一段文字"

# 用单引号包裹——效果完全一样
'这也是一段文字'
```

只要前后引号类型一致就可以：双引号配双引号，单引号配单引号。

### print() 和字符串的组合

把字符串放进 `print()` 的括号里，它就会被显示出来：

```python
# 语法结构：
print("你想说的话")
```

括号内、引号内的内容，Python 会原样输出到屏幕上。

### 引号本身不会出现在输出中

`print()` 输出的只是引号**里面**的内容，引号本身不会显示。

```
你好
```

引号是给 Python 看的标记，不是输出的一部分。

### 每次 print() 自动换行

每调用一次 `print()`，输出会在末尾自动换行。如果你想输出多行内容，就写多次 `print()` 调用——每次输出各自独立的一行。

```python
# 第一次调用
print("……")

# 第二次调用
print("……")
```

两个 `print()` 会产生两行输出，先后排列，之间自动带着换行。

### 双引号和单引号可以互换

```python
print("你好")   # 双引号版本
print('你好')   # 单引号版本——输出完全一样
```

两种写法等价，但前后必须匹配：`"你好'` 这样混搭是不行的。

## 常见错误

### 错误 1：忘记引号

```python
print(你好)
```

运行后会报错 —— Python 会把 `你好` 当成一个变量名去查找，但并没有这个变量，所以会提示 `NameError`。**任何文本内容都必须用引号包裹。**

### 错误 2：使用中文引号

```python
print("你好")
```

中文引号 `""` 和英文引号 `""` 长得像但不是同一个字符。Python 只识别英文引号。务必使用英文输入法下的双引号或单引号。

### 错误 3：引号不匹配

```python
print("你好')
```

开头用了双引号，结尾却用了单引号——Python 会认为字符串还没结束，导致语法错误。务必保持前后一致。

```
你好，Python！
```

**函数签名：**

```python
def py_learn():
```

本题不需要参数，也不需要返回值。只需要在函数体内使用 `print` 打印出目标字符串即可。
