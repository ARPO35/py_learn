# 检查元素是否在列表中

老师点名时，会在名单上找学生的名字——如果名字在名单上，就回答「到」；不在，就没人回应。Python 里检查一个元素「在不在」列表中，用的就是 `in` 操作符。

## 概念讲解

`in` 操作符用来判断一个值是否包含在列表中。它返回一个布尔值：
- 如果元素在列表中，返回 `True`
- 如果元素不在列表中，返回 `False`

这是 Python 中最常用的存在性检查方式，写起来非常自然，就像读英语句子一样。

## 代码示例

```python
# 判断数字是否在列表中
nums = [1, 2, 3, 4, 5]
print(3 in nums)   # True
print(8 in nums)   # False

# 字符串列表也一样
fruits = ["苹果", "香蕉", "橘子"]
print("香蕉" in fruits)  # True
print("西瓜" in fruits)  # False
```

**列表切片（扩展知识）**：切片可以取出列表的一部分，语法为 `列表[开始:结束]`，注意结束索引不包含在内。

```python
nums = [10, 20, 30, 40, 50]
print(nums[1:3])   # [20, 30]  取索引 1 到 2
print(nums[:3])    # [10, 20, 30]  从头到索引 2
print(nums[2:])    # [30, 40, 50]  从索引 2 到末尾
```

## 常见错误

- `in` 只能判断单个元素是否在列表中，不能判断一个子列表是否「整体包含」在列表中。例如 `[1, 2] in [1, 2, 3]` 是 `False`（因为列表里没有 `[1, 2]` 这个元素，只有三个整数）。
- 不要混淆 `in` 和 `==`：`in` 检查「包含关系」，`==` 检查「相等关系」。

## 题目要求

编写函数 `py_learn(numbers, target)`，接收一个列表 `numbers` 和一个目标值 `target`，返回一个布尔值表示 `target` 是否在 `numbers` 中。

- 输入：`numbers`（list）—— 任意元素组成的列表；`target` —— 要查找的值
- 输出：`bool`——`True` 表示存在，`False` 表示不存在

## 提示

一行代码即可完成：`return target in numbers`
