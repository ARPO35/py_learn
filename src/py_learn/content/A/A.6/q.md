# 练习 A.6：布尔值与比较

想象一个开关——只有「开」和「关」两种状态。Python 中也有类似的数据类型叫**布尔类型**（`bool`），它的值只有两个：`True`（真/成立）和 `False`（假/不成立）。布尔值是程序做决策的基础。

## 概念讲解

**布尔类型**是 Python 中最简单的数据类型之一。它只有两个可能的值：
- `True` — 表示条件成立
- `False` — 表示条件不成立

布尔值通常由**比较运算符**产生。比较运算符用来判断两个值之间的关系：

| 运算符 | 含义 | 示例 |
|--------|------|------|
| `==` | 等于 | `3 == 3` → `True` |
| `!=` | 不等于 | `3 != 4` → `True` |
| `>` | 大于 | `10 > 5` → `True` |
| `<` | 小于 | `10 < 5` → `False` |
| `>=` | 大于等于 | `5 >= 5` → `True` |
| `<=` | 小于等于 | `6 <= 5` → `False` |

比较运算的结果**永远是** `True` 或 `False`。

## 代码示例

```python
# 比较数字大小
result = 15 > 10
print(result)     # 输出：True

# 判断相等
print(3 == 4)     # 输出：False

# 判断年龄是否达到成年标准
age = 20
is_adult = age >= 18
print(is_adult)   # 输出：True
```

```python
# 混合使用变量和比较
score = 85
passed = score >= 60
print("是否及格：", passed)  # 输出：是否及格： True

# 不等于的用法
print(10 != 10)   # 输出：False（10 等于 10，所以不等于不成立）
print(10 != 11)   # 输出：True
```

## 常见错误

**混淆 `=` 和 `==`**
- `=` 是**赋值**运算符，用来把值放入变量：`x = 5`
- `==` 是**比较**运算符，用来判断两个值是否相等：`x == 5` → `True`

初学者经常在 `if` 语句或 `return` 里误写 `=`，导致语法错误。记住：**一个等号赋值，两个等号判断**。

## 题目要求

写一个函数 `py_learn(age)`，接收一个整数 `age`（年龄）。如果 `age >= 18`（成年），返回 `True`；否则返回 `False`。

**函数签名：** `def py_learn(age):`

**返回值：** `bool` 类型（`True` 或 `False`）

## 提示

- 直接用 `return age >= 18` 即可——比较运算的结果就是布尔值
- 不需要写 `if age >= 18: return True else: return False`，那样太啰嗦了
