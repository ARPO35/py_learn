# 练习 A.6：字符串回文判断

## 题目说明

编写函数 `py_learn`，接收一个字符串 `s`，判断它是否为回文字符串。回文字符串是指正着读和反着读一样的字符串（忽略大小写）。

## 示例

```python
py_learn("level")     # 返回 True
py_learn("python")    # 返回 False
py_learn("Racecar")   # 返回 True（忽略大小写）
```

## 提示

- 先将字符串转为小写：`s.lower()`
- 使用字符串切片反转：`s[::-1]`
- 比较原字符串和反转后的字符串是否相等

## 要求

- 函数签名：`def py_learn(s):`
- 忽略大小写
- 返回值类型：`bool`
