# 练习 A.3：条件判断

## 题目说明

编写函数 `py_learn`，接收一个整数 `score`（0-100 的分数），根据分数返回对应的等级：

- 90 分及以上：返回 `"优秀"`
- 80-89 分：返回 `"良好"`
- 60-79 分：返回 `"及格"`
- 60 分以下：返回 `"不及格"`

## 示例

```python
py_learn(95)  # 返回 "优秀"
py_learn(85)  # 返回 "良好"
py_learn(65)  # 返回 "及格"
py_learn(45)  # 返回 "不及格"
```

## 提示

- 使用 `if`、`elif`、`else` 进行条件判断
- 比较运算符：`>=`（大于等于）、`<`（小于）
- 使用 `return` 返回结果字符串

## 要求

- 函数签名：`def py_learn(score):`
- 返回值类型：`str`
