# A.12 for 循环入门

数零钱罐里的硬币时，你会一枚一枚拿出来，每拿一枚就加到一个累计数上。Python 中的 for 循环就像这个重复动作——它自动帮你「逐个取出」列表中的元素，然后你可以对每个元素做同样的操作。

## 概念讲解

`for` 循环是 Python 中遍历（逐一处理）序列的最常用方式。它的语法非常直观：

```python
for 变量 in 序列:
    要重复执行的代码
```

Python 会从序列中依次取出每个元素，赋值给「变量」，然后执行缩进中的代码块。当所有元素都取完后，循环自动结束。

一个特别常见的模式是**累加（accumulation）**：先设置一个初始值（通常为 0），然后在循环中逐个加上每个元素的值。

## 代码示例

**示例 1：遍历一个列表并打印每个元素**

```python
fruits = ["苹果", "香蕉", "橘子"]
for fruit in fruits:
    print(fruit)
```

输出：
```
苹果
香蕉
橘子
```

每次循环，`fruit` 依次被赋值为 `"苹果"`、`"香蕉"`、`"橘子"`，然后执行循环体内的 `print(fruit)`。

**示例 2：累加求和**

```python
numbers = [10, 20, 30]
total = 0         # 初始化为 0
for num in numbers:
    total += num  # 每次把当前元素加到 total 上
print(total)
```

第一次循环：`total = 0 + 10 = 10`  
第二次循环：`total = 10 + 20 = 30`  
第三次循环：`total = 30 + 30 = 60`  
最终输出 `60`。

## 常见错误

1. **忘记初始化累加变量**：如果直接写 `total += num`，而 `total` 没有事先赋值，Python 会报 `NameError`，因为 `total` 未定义。一定要先写 `total = 0`。

2. **缩进错误**：把 `total += num` 写在了 `for` 语句的缩进外面，导致累加只执行最后一次（或根本不执行）。记住：循环体必须比 `for` 多一个缩进。

编写函数 `py_learn(numbers)`：

- **输入**：一个整数列表 `numbers`
- **输出**：返回列表中所有元素的总和（整数）
- **示例**：`py_learn([1, 2, 3])` 返回 `6`

## 提示

- 先初始化一个变量 `total = 0`，用来记录累加结果。
- 用 `for` 循环逐个遍历列表中的每个元素。
- 在循环体内用 `total += num`（或 `total = total + num`）累加。
- 循环结束后，`return total`。
