# 练习 A.8：多分支条件 elif

考试成绩出来了，老师要根据分数划等级：90 分以上是「优秀」，80 分以上是「良好」，60 分以上是「及格」，其余的「不及格」。如果只用 `if/else` 只能处理两种情况，而现实往往需要更多的分支判断。

## 概念讲解

`elif` 是 `else if` 的缩写，用于在 `if` 和 `else` 之间插入多个条件分支。Python 会从上到下依次判断每个条件，一旦遇到 `True` 的条件，就执行对应的代码块，并跳过后续所有分支。

格式如下：

```python
if 条件1:
    条件1为真时执行
elif 条件2:
    条件1为假、条件2为真时执行
elif 条件3:
    条件1和2均为假、条件3为真时执行
else:
    以上条件全为假时执行
```

**条件顺序至关重要**：应从最严格（值最高）的条件开始判断，逐步放宽。因为一旦某个条件满足，后续分支就不再执行。

## 代码示例

**示例 1：成绩等级评定**

```python
score = 85
if score >= 90:
    result = "优秀"
elif score >= 80:
    result = "良好"
elif score >= 60:
    result = "及格"
else:
    result = "不及格"
print(result)  # 输出：良好
```

分数 85 不满足 `>= 90`，进入第一个 `elif`，满足 `>= 80`，所以结果为「良好」。

**示例 2：不同分数对应的结果**

```python
for s in [95, 85, 65, 45]:
    if s >= 90:
        print(s, "→ 优秀")
    elif s >= 80:
        print(s, "→ 良好")
    elif s >= 60:
        print(s, "→ 及格")
    else:
        print(s, "→ 不及格")
# 输出：
# 95 → 优秀
# 85 → 良好
# 65 → 及格
# 45 → 不及格
```

## 常见错误

**错误 1：条件顺序颠倒**

```python
# 这样写会导致所有 >=60 的分数都输出"及格"而非"优秀"或"良好"
if score >= 60:
    result = "及格"
elif score >= 80:
    result = "良好"
elif score >= 90:
    result = "优秀"
else:
    result = "不及格"
```

因为 `>= 60` 是最宽的条件，95 分也会先匹配它，后面的 `elif` 永远不会执行。

**错误 2：用多个独立的 `if` 代替 `elif`**

```python
# 每个 if 都会判断，结果会被后面的覆盖
if score >= 90:
    result = "优秀"
if score >= 80:
    result = "良好"   # 85 分到这里会把"优秀"覆盖掉
```

## 题目要求

编写函数 `py_learn`，接收一个整数参数 `score`（0-100），按照以下规则返回对应的等级：

- `score >= 90` → 返回 `"优秀"`
- `score >= 80` → 返回 `"良好"`
- `score >= 60` → 返回 `"及格"`
- 否则 → 返回 `"不及格"`

函数签名：`def py_learn(score):`
返回值类型：`str`

## 提示

- 从最高分条件（`>= 90`）开始写，逐级降低
- 用 `if`、`elif`、`elif`、`else` 四级结构
- 不需要处理分数超出 0-100 的情况
