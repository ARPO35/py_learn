# 练习 A.14：while 循环

想象你在拆一个用乐高积木搭成的数字——从最后一块开始拆，每拆一块就数一数它的点数，直到所有积木都拆完。while 循环就是这种「边拆边数，直到拆光」的重复动作。

## 概念讲解

**while 循环**是 Python 中最基本的循环结构之一。它反复执行一段代码，只要一个条件保持为真（True）。基本语法：

```python
while 条件:
    代码块
```

每次执行完代码块后，Python 会重新检查条件——如果仍然为真，就再执行一遍；如果变为假，就退出循环继续执行后面的代码。

计算一个整数的各位数字之和，正好可以用 while 循环来实现。核心操作是两个：

- `n % 10`：取 n 的个位数字（模运算）
- `n // 10`：去掉 n 的个位数字（整除）

例如 n=123：
1. 123 % 10 = 3（取个位），累加；然后 123 // 10 = 12（去掉个位）
2. 12 % 10 = 2（取个位），累加；然后 12 // 10 = 1（去掉个位）
3. 1 % 10 = 1（取个位），累加；然后 1 // 10 = 0（去掉个位）
4. n=0，循环结束

最终 3+2+1=6。

## 代码示例

```python
# 示例 1：计算 123 的各位数字之和
n = 123
total = 0
while n > 0:
    total += n % 10
    n = n // 10
print(total)  # 输出 6
```

```python
# 示例 2：计算 4567 的各位数字之和
n = 4567
total = 0
while n > 0:
    total += n % 10
    n = n // 10
print(total)  # 输出 22
```

## 常见错误

1. **忘记更新循环变量**：如果漏掉 `n = n // 10` 这一行，n 的值永远不会变化，循环条件 `n > 0` 永远为真——程序将无限运行下去（死循环）。

2. **条件写反**：如果把 `while n > 0` 写成 `while n < 0`，循环一次都不会执行。
