# 练习 A.10：列表入门

你去超市购物时会把想买的东西写在纸条上，一条一条按顺序记好。Python 里的 **列表（list）** 就是这样的「购物纸条」——它可以按顺序存放多个数据，想加就加，想看就看。

## 概念讲解

列表是用方括号 `[]` 包裹的一组数据，数据之间用逗号分隔。列表里的每一项称为**元素（element）**，元素可以是数字、字符串，甚至是另一个列表。

```python
nums = [10, 20, 30]          # 整数列表
names = ["小明", "小红", "小刚"]  # 字符串列表
mix = [1, "hello", 3.14]     # 不同类型也可以
```

我们可以通过**索引（index）**来访问列表中的某个元素。索引从 `0` 开始计数——`0` 代表第一个元素，`1` 代表第二个，以此类推。Python 还支持**负数索引**，`-1` 表示最后一个元素，`-2` 表示倒数第二个。

`len()` 函数可以获取列表的长度（即元素个数）。

## 代码示例

### 示例一：创建并访问列表

```python
fruits = ["苹果", "香蕉", "橘子"]
print(fruits[0])   # 输出：苹果
print(fruits[1])   # 输出：香蕉
print(fruits[-1])  # 输出：橘子（最后一个）
print(len(fruits)) # 输出：3
```

### 示例二：用索引修改元素

```python
scores = [90, 85, 88]
scores[1] = 95     # 把第二个元素从 85 改为 95
print(scores)      # 输出：[90, 95, 88]
```

### 示例三：空列表

```python
empty = []
print(len(empty))  # 输出：0
# empty[0] ← 错误！空列表没有元素，访问会报错
```

## 常见错误

1. **索引越界（IndexError）**：访问了不存在的索引位置。比如列表只有 3 个元素，却去访问 `numbers[3]`（有效索引是 0、1、2）或 `numbers[100]`。记住：长度为 `n` 的列表，有效索引范围是 `0` 到 `n-1`（正数）以及 `-1` 到 `-n`（负数）。
2. **混淆长度和索引**：`len(numbers)` 返回元素个数，而最后一个元素的索引是 `len(numbers) - 1`，不是 `len(numbers)`。

## 题目要求

编写函数 `py_learn`，接收一个整数列表 `numbers`。

- 如果列表为空，返回字符串 `"列表为空"`
- 否则返回格式化的字符串：`"第一个：{第一个元素}，最后一个：{最后一个}，共{长度}个"`

**示例：**

```python
py_learn([1, 2, 3])       # 返回 "第一个：1，最后一个：3，共3个"
py_learn([100])           # 返回 "第一个：100，最后一个：100，共1个"
py_learn([])              # 返回 "列表为空"
```

## 提示

- 先判断列表是否为空（`if not numbers:` 或 `if len(numbers) == 0:`）
- 取第一个元素用 `numbers[0]`
- 取最后一个元素用 `numbers[-1]`
- 获取长度用 `len(numbers)`
- 用 f-string 格式化输出：`f"第一个：{...}，最后一个：{...}，共{...}个"`
