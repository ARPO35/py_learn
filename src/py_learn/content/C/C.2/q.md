# 练习 C.2：字典字符计数

## 题目说明

编写函数 `py_learn`，接收一个字符串 `text`，统计其中每个字符出现的次数，返回一个字典。

只统计字母和数字，忽略空格、标点符号等。字母不区分大小写（先转为小写）。

## 示例

```python
py_learn("Hello")           # 返回 {"h": 1, "e": 1, "l": 2, "o": 1}
py_learn("abab")            # 返回 {"a": 2, "b": 2}
py_learn("123 456")         # 返回 {"1": 1, "2": 1, "3": 1, "4": 1, "5": 1, "6": 1}
```

## 提示

- 先将字符串转为小写：`text.lower()`
- 使用 `char.isalnum()` 判断字符是否为字母或数字
- 遍历字符串用 `for` 循环
- 用字典计数：`result[char] = result.get(char, 0) + 1`

## 要求

- 函数签名：`def py_learn(text):`
- 只统计字母和数字（使用 `isalnum()`）
- 不区分大小写
- 返回值类型：`dict`
