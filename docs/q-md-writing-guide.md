# q.md 教学材料编写规范 v2

## 核心原则

**禁止包含任何完整、可直接复制粘贴运行的代码块。**

教学材料的代码只用于展示**语法结构**和**原理**，不提供可运行的完整程序。学习者的任务是理解原理后自己写出代码。

## 代码块判定标准

### 禁止（完整可运行代码块）

满足以下**任一条件**即视为完整代码块：

1. **完整脚本**：包含变量定义 → 操作逻辑 → 输出/返回的完整流程
2. **完整函数体**：函数体内有完整的业务逻辑实现（不含 `pass`）
3. **可独立运行的多行片段**：复制到 `.py` 文件可直接 `python xxx.py` 运行

禁止示例：

```python
# 禁止：完整的 for 循环脚本
numbers = [1, 2, 3, 4, 5]
total = 0
for num in numbers:
    total += num
print(total)
```

```python
# 禁止：完整的 if/else 脚本
score = 85
if score >= 60:
    print("及格")
else:
    print("不及格")
```

```python
# 禁止：完整的函数实现（含解题逻辑）
def is_prime(n):
    if n < 2:
        return False
    for i in range(2, n):
        if n % i == 0:
            return False
    return True
```

### 允许（语法碎片、原理讲解）

允许以下形式的代码片段：

**1. 语法结构模板（含占位符）**

```python
for 变量 in 可迭代对象:
    循环体
```

**2. 单行/两行概念演示**

```python
numbers[0]   # 取列表第一个元素
numbers[-1]  # 取列表最后一个元素
```

**3. 孤立语法片段（不构成完整程序）**

```python
# 展示 for 循环如何遍历——只展示循环头和一行体，不给完整逻辑
for fruit in fruits:
    # 这里 fruit 依次取每个元素
```

**4. 错误示例（不可运行）**

```python
print(你好)    # 错误！忘记加引号
print("你好")   # 正确
```

**5. 输出效果展示（标注为「输出」）**

```
你好，Python！
```

**6. 局部原理演示（不构成完整逻辑）**

```python
# 展示取余运算
7 % 2   # 结果是 1（7 除以 2 余 1）
8 % 2   # 结果是 0（8 除以 2 余 0，说明是偶数）
```

**7. 分步拆解（每步独立，不连成完整程序）**

```python
# 步骤一：初始化
total = 0

# 步骤二：循环遍历（此处不写完整循环体）
```

```python
# 步骤三：累加单个元素
total = total + num   # 或 total += num
```

## 各练习类型的改写指南

### 条件判断（if/else）

错误做法——展示完整判断：
```python
score = 85
if score >= 60:
    print("及格")
else:
    print("不及格")
```

正确做法——拆解结构：
```python
# if 语句的结构：
if 条件:
    条件成立时执行的代码
else:
    条件不成立时执行的代码
```

```python
# 条件写成 score >= 60 —— 判断分数是否大于等于 60
# 条件为 True 时走 if 分支
# 条件为 False 时走 else 分支
```

### 循环（for/while）

错误做法——展示完整累加：
```python
total = 0
for num in numbers:
    total += num
```

正确做法——拆解每一步：
```python
# for 循环的语法：
for 临时变量 in 序列:
    # 每次循环，临时变量取序列中的下一个元素
```

```python
# 累加模式需要先准备一个变量：
total = 0  # 初始值为 0
```

```python
# 每次循环中更新：
total = total + 当前元素  # 把当前元素加到累计值上
```

### 函数定义

错误做法——给出完整函数体：
```python
def py_learn(celsius):
    fahrenheit = celsius * 9 / 5 + 32
    return fahrenheit
```

正确做法——分步讲解：
```python
# 函数定义语法：
def 函数名(参数列表):
    函数体
    return 返回值
```

```python
# 计算华氏温度的公式拆解：
# 1. 乘以 9/5（或 1.8）
# 2. 加上 32
# 这两步可以合并为一行：celsius * 9 / 5 + 32
```

### 列表操作

错误做法：
```python
fruits = ["苹果", "香蕉", "橘子"]
print(fruits[0])
print(fruits[-1])
print(len(fruits))
```

正确做法：
```python
# 创建列表：
["苹果", "香蕉", "橘子"]  # 方括号，元素用逗号分隔
```

```python
# 索引访问：
fruits[0]   # 第一个元素（索引从 0 开始）
fruits[-1]  # 最后一个元素（负数从末尾倒数）
```

```python
# 获取长度：
len(列表)  # 返回列表中元素的个数
```

## 教学结构规范

每个 q.md 保持以下结构：

1. **标题**：`# 练习 A.N：标题`
2. **生活类比**（1-2 句引入）
3. **概念讲解**：用通俗语言解释新概念（不含完整代码）
4. **语法拆解**：逐个拆解语法点，每段只展示 1-3 行代码片段
5. **常见错误**：错误代码 + 解释（错误代码可以是完整片段，因为不能直接运行来通过练习）
6. **题目要求**：明确函数签名和需求
7. **提示**：给思路不给代码
